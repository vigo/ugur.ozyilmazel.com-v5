JS İstanbul 2012

28 Ekim 2012, Pazar 12:25
jstanbul javascript

Yaklaşık 3 ay’lık bir gecikme ardından JS İstanbul 2012 ile ilgili yazımı yazmaya fırsat bulabildim!

28 - 29 Temmuz 2012’de Bahçeşehir Üniversitesi, Beşiktaş kampüsünde düzenlenen etkinliğe katılım tahminlerin üzerinde oldu. Cumartesi - Pazar günlerine gelen etkinliğin sadece ilk gününe katılabildim.

Sabah saat 10:30’daki açılış konuşmasının yapıldığı salon tıklım tıklım doluydu. Tahminimce 200+ kişi vardı. Hatta o esnada yanımda oturan python programcısı arkadaşım Metin Amiroff:

Acaba PythonCon yapsak bu kadar kişi gelirmi?

diye sordu… Ben de **biraz zor…** dedim…

JavaScript, pek çok farklı alandaki insana hitap ediyor:

  • Grafik tasarımcısı
  • Backend programcı (php, python, ruby, java, dotnet vs…)
  • Frontend (html, css, js)

Yani işi WEB olan herkesin yolu bir şekilde JavaScript’le kesişmek zorunda!

Konuşmacılar

Yaklaşık 17 kişi vardı. İşin güzel yanı, hiç bir sunum / konuşma diğeriyle çakışmıyordu. Bu bakımdan etkinliğin iki güne yayılması izleyici açısından çok iyi birşey… Genelde zaman ve mekan kısıtlamaları yüzünden, şahsen ben, tercih yapmak zorunda kalmaktan nefret ediyorum. İlgimi çeken iki konuya da girmek istiyorum ama birini seçiyorum. İşte bu etkinlikte böyle bir durum olmadı!

Etkinliğin önemli konuklarından biri, taaa Amerika’dan gelen, Backbone.js’in core yazılımcılarından Tim Branyen’di. Diğer konuşmacılarsa;

  • Disqus’tan Burak Yiğit Kaya JS ve REST Uyumlu API İletişimi
  • Rocket GmbH’dan Osman Yüksel Jasmine ile Behavior Driven Development
  • Social Wire’dan Volkan Özçelik JavaScript ile Widget Geliştirmek
  • Markafoni’den Fatih Kadir Akın Backbone Fundamentals
  • Fatih Hayrioğlu CSS3’ün JavaScript’ten Rol Çalması
  • Emrah Olgun Yazılım evlerinde Ön Yüz Geliştiricileri Konumlamak
  • Armağan Amcalar tartJS ile Büyük Ölçekli Uygulama Geliştirme
  • Kadir Pekel Node, Connect ve Meryl ile Web Uygulama Geliştirme
  • Burcu Doğan Node Ortamının Mimarisi
  • Altan Yılmaz Javascript Mobile Applications
  • Fatih Acet JavaScript Debugging Yöntemleri
  • Emrah Ayanoğlu Node.js ve Socket.io ile Mutliplayer HTML5 Oyun Geliştirme
  • Serdar Akarca Knockout.js ve Sammy.js ile Uygulama Geliştirme
  • Emre Evren Yalçın JavaScript Injections
  • Sinan Taga Javascript - Nereden Nereye, Kısa bir tarihçe
  • Emir Karşıyakalı Jstanbul.org Açılış Konuşması

Ülkemizde bu tür etkinlikleri görmek gurur veriyor. Özellikle arkasına büyük kurumları almadan büyük işlere girişmek herkesin yapabileceği birşey değil. Katıldığım ilk günden çok memnun ayrıldım. Hem girdiğim sunular hem de öğle arasında arkadaşlarla yaptığımız muhabbetler çok zevkliydi.

JS İstanbul 2013’ü iple çekiyorum!
