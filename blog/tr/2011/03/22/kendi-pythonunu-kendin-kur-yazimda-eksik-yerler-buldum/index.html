Herşey güzel güzel ilerledi. Sıra python’un PIL paketini kurmaya gelmişti ki jeton düştü! Python Image Library’nin ihtiyacı olan 2 kütüphane eksikti… bunlar:

jpeg
freetype

Bu kütüphaneleri ports’dan kurmak gerekiyor öncesinden:

sudo port install jpeg freetype
